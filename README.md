## 📖 Про проєкт

**Politime Launcher** — це сучасний, швидкий та зручний лаунчер Minecraft, розроблений спеціально для комфортної гри на сервері **Politime**. Лаунчер позбавляє потреби вручну завантажувати моди, налаштовувати Java чи Forge/NeoForge — усе відбувається в один клік!

## ✨ Основні функції та можливості

- ⚡ **Запуск у 1 клік:** Автоматичне завантаження потрібної версії Minecraft (1.21.1) та ядра NeoForge.
- 📦 **Гнучкий вибір та налаштування збірок:**
  - **Lite збірка** — мінімальний набір модів для слабких ПК.
  - **Повна збірка** — повна збірка з графічними покращеннями та максимумом контенту.
  - **Налаштування збірки** — можливість персоналізувати та налаштувати збірку під власні потреби.
- 🔄 **Авто-оновлення:** Клієнт лаунчера та моди синхронізуються з сервером при кожному запуску.
- ☕ **Автоматичне встановлення Java 21:** Лаунчер сам перевіряє наявність потрібної версії Java і може завантажити її за потреби.
- 🎮 **Discord Rich Presence:** Показує статус гри, обрану збірку та кількість гравців онлайн у вашому профілі Discord.
- 📰 **Новини проєкту:** Вбудована стрічка новин з підтримкою форматування Markdown та зображень.
- 🎵 **Атмосферний супровід:** Фонова музика та звукові ефекти UI із можливістю налаштування гучності.
- ⚙️ **Налаштування ресурсів:** Просте виділення оперативної пам'яті (RAM) та вибір шляху до Java.

## ☕ Рекомендації щодо Java

> [!IMPORTANT]
> ⚠️ **НАСТІЙНО РЕКОМЕНДУЄТЬСЯ ВЛАСНОРУЧ ВСТАНОВИТИ JAVA 21 ПЕРЕД ВИКОРИСТАННЯМ ЛАУНЧЕРА!**  
> Для досягнення максимальної стабільності, відсутності затримок та запобігання збоям переконайтеся, що ви завантажили та встановили **Java 21 (64-bit)** (наприклад, [Adoptium Temurin 21 64-bit](https://adoptium.net/temurin/releases/?version=21)) на свій ПК перед першим запуском лаунчера.

| Версія Minecraft | Рекомендована версія Java | Примітка |
| :--- | :--- | :--- |
| **Minecraft 1.21+ (NeoForge)** | **Java 21 (64-bit)** ⚡ | **Встановити вручну перед запуском** |
| **Minecraft 1.17 – 1.20.4** | **Java 17 (64-bit)** | Для старіших версій гри |
| **Minecraft 1.12.2 та нижче** | **Java 8 (64-bit)** | Класична Java для старих збірок |

> [!NOTE]
> **Автоматичний режим:** Якщо Java 21 не знайдено на системі, Politime Launcher спробує завантажити її автоматично, проте для найкращої продуктивності рекомендуємо ручне встановлення.

> [!TIP]
> Переконайся, що в налаштуваннях лаунчера виділено щонайменше **4-6 ГБ оперативної пам'яті (RAM)**.

## 📥 Завантаження та встановлення

<div align="center">

[![Завантажити LauncherSetup.exe](https://img.shields.io/badge/Завантажити-LauncherSetup.exe-brightgreen?style=for-the-badge&logo=windows&logoColor=white)](https://github.com/wolfenhteyn/minecraft-launcher-assets/releases/latest/download/LauncherSetup.exe)

</div>

### Кроки для запуску:
1. **Встановіть Java 21:** Настійно рекомендується завантажити та встановити [Java 21 64-bit (Adoptium Temurin)](https://adoptium.net/temurin/releases/?version=21) перед використанням лаунчера.
2. **Завантажте лаунчер:** Натисніть кнопку **[Завантажити]** вище або перейдіть у розділ [Releases](https://github.com/wolfenhteyn/minecraft-launcher-assets/releases/latest).
3. **Встановіть лаунчер:** Запустіть файл `LauncherSetup.exe` та виберіть директорію інсталяції.
4. **Налаштуйте збірку:** Вкажіть ігровий нікнейм, оберіть бажану збірку модів, налаштуйте її під себе та виділіть обсяг RAM.
5. Натисніть **«Грати»** та насолоджуйтесь грою!

## 🛠️ Для розробників (Збірка з вихідного коду)

Якщо ви бажаєте зібрати лаунчер із вихідного коду:

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/wolfenhteyn/minecraft-launcher-assets.git
   cd minecraft-launcher-assets/source-code
   ```

2. **Встановіть залежності:**
   ```bash
   npm install
   ```

3. **Запустіть у режимі розробки:**
   ```bash
   npm run dev
   ```

4. **Зберіть інсталятор для Windows:**
   ```bash
   npm run build
   ```
